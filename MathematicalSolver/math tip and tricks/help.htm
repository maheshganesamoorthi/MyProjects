1. Constant

Circumferential rate pi is defined. pi=3.1415926. You can input pi directly in your expression, for example, 2pi*sin(x).

2. Operations

Math Calculator recognizes the following arithmetic operations:
    "+" - addition
    "-" - subtraction
    "*" - multiplication
    "/" - division
    "^" - exponentiation
Example: 2^3+3*1-8/4=9

3. The Priority Of Operations And Functions

The priority of operations in Math Calculator is:
      Functions
      ^
      negative(-)
      *,/
      +,-
Functions have the largest priority; "+", "-" - the smallest. 
You can use parentheses to group operations with lower precedence 
to be calculated before operations with higher precedence.
4. Functions
Math Calculator allows to use the following functions:
   abs       absolute value
   atn        inverse tangent
   cos       cosine
   exp       exponent (exp(x) - e raised to the power x)
   int         rounds number to the closest integer
   log        natural logarithm
   rnd        random number
   sin        sine
   sqr        square root
   tan        tangent
   sec       Secant             Sec(X)=1/Cos(X)
   csec     Cosecant         Cosec(X)=1/Sin(X)
   ctan      Cotangent        Cotan(X)=1/Tan(X)
   asin       inverse sine    Arcsin(X)=Atn(X/Sqr(-X*X+1))
   acos      inverse cosine            Acos(X)=Atn(-X/Sqr(-X*X+1))+2*Atn(1)
   asec      inverse secant            Asec(X)=Atn(X/Sqr(X*X-1))+Sgn((X)-1)*(2*Atn(1))
   acsec     inverse cosecant       Acsec(X)=Atn(X/Sqr(X*X-1))+(Sgn(X)-1)*(2*Atn(1))
   actan     inverse tangent           Actan(X)=Atn(X)+2*Atn(1)
   sinh       Hyperbolic Sine           SinH(X)=(Exp(X)-Exp(-X))/2
   cosh      Hyperbolic Cosine       CosH(X)=(Exp(X)+Exp(-X))/2
   tanh       Hyperbolic Tangent     TanH(X)=(Exp(X)-Exp(-X))/(Exp(X)+Exp(-X))
   ctanh     Hyperbolic Cotangent         CtanH(X)=(Exp(X)+Exp(-X))/(Exp(X)-Exp(-X))
   sech     Hyperbolic Secant               SecH(X)=2/(Exp(X)+Exp(-X))
   csech   Hyperbolic Cosecant           CsecH(X)=2/(Exp(X)-Exp(-X))
   asinh     Inverse Hyperbolic Sine      AsinH(X)=Log(X+Sqr(X*X+1))
   acosh    Inverse Hyperbolic Cosine         AcosH(X)=Log(X+Sqr(X*X-1))
   atanh     Inverse Hyperbolic Tangent        AtanH(X)=Log((1+X)/(1-X))/2
   asech    Inverse Hyperbolic Secant         AsecH(X)=Log((Sqr(-X*X+1)+1)/X)
   acsech  Inverse Hyperbolic Cosecant     AcsecH(X)=Log((Sgn(X)*Sqr(X*X+1)+1)/X)
   actanh   Inverse Hyperbolic Cotangent    ActanH(X)=Log((X+1)/(X-1))/2

5. Syntax
The following expressions are wrong:
    sinx+logy
    xy
    pitan(x)
The following expressions are right:
    sin(x)+log(y)
    x*y
    pi*tan(x)
    2pi(1+sin(2x))
    2*pi*(1+sin(2*x))
    sin(x)x
    sin(x)3
    sin(x)cos(x)

6. Expression Calculating
To calculate expression, following these steps:
    1> Click Expression button
    2> Input your expression in Expression field
    3> Input the value of x in x0 field
    4> Click Calculate button
    You can input an expression including variable x, for example, log(x), then input a value in x0 field. 
    You can also input an expression such as log(20) directly in Expression field.

7. Solving Equation
To solve  equation, following these steps:
    1> Click Root button
    2> Input your equation in Expression field
    3> Input the interval values of x in x From field and To field
    4> Click Calculate button

8. Finding Extremum
To find  extremums of a function, following these steps:
    1> Click Extremum button
    2> Input your function in Expression field
    3> Input the interval values of x in x From field and To field
    4> Click Calculate button

9. Derivative Calculating
To calculate  derivative of a function, following these steps:
    1> Click Derivative button
    2> Input your function in Expression field
    3> Input the value of x in x0 field
    4> Click Calculate button

10. 2nd Order Derivative Calculating
To calculate 2nd order derivative of a function, following these steps:
    1> Click Derivative 2 button
    2> Input your function  in Expression field
    3> Input the value of x in x0 field

11. Integral Calculating
To calculate  denifite integral of a function, following these steps:
    1> Click Integral button
    2> Input your function in Expression field
    3> Input the interval values of x in Lower Limit Of x field and Upper Limit Of x field
    4> Click Calculate button
